## Docker Hub 에서 이미지 다운로드 CLI
```bash
docker pull ubuntu:24.04
```
- `docker` : 도커 명령어 
- `pull` : 받다
- `ubuntu` : 이미지 이름
- `:` : 이미지 이름과 태그 구분자
- `24.04` : 태그 이름

## Docker 이미지 확인

```bash
docker images
```
